# Product Requirements Document

See the user-provided PRD in the project request for the full Tenant Cost Command Center requirements. This repository implements the MVP scaffold described there:

- Multi-tenant EKS and directly tagged AWS service cost allocation.
- Tenant, product-line, platform, and executive dashboards.
- Budget, anomaly, recommendation, tagging compliance, allocation rule, and refresh status workflows.
- AWS-native ingestion and infrastructure extension points.

## MVP Definition

The MVP is a showback-focused portal that uses mock repositories first so dashboard, API, and allocation behavior can be validated before AWS integration.
